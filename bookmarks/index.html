(Maybe) Useful Bookmarks

Social Media

https://bsky.app/
Bluesky

Bluesky is basically Twitter but without Elon Musk. The main reason I use Blusky is the ease of finding a community to become a part of.

https://discord.com/
Discord

Discord is made for people to be part of smaller communities. There are communities (servers) for programming, gaming, hiking... everything.

https://logiverse.social/
Logiverse

Logiverse is a HIGHLY EXPERIMENTAL network of social media sites. It's built on Tadi Web, which means its really simple.

Some sites on Logiverse
  • Loggo by Sven Lammertink
  • Login by Lu Wilson
  • Login Redux by Rosy
  • Logon by Rafał Hirsz
  • Lodge by Mae Moon
  • Blogin by blag
  • Logarithm by Tristie
  • Lofy by Mae Moon
  • Signup by gaimerI17
  • Logiverse Café by Oleksandr Kryvonos

https://www.pondiverse.com/
Pondiverse

Pondiverse is two things: 1. A way to share art 2. A way to share tools to create art.

https://mas.to/
Mastodon

Mastodon is a free and open-source software platform for decentralized social networking with microblogging features similar to Twitter.

https://statusphere.mozzius.dev/
Statusphere

Statusphere is an experimental social media built with ATProto.

Tools

https://jsoneditoronline.org/
JSON Editor

This is the best online JSON editor I've found. It's fairly easy to use and offers everything needed.

https://mcbe-essentials.github.io/
MCBE Essentials

I play Minecraft (specifically Bedrock Edition) surprisingly lot. This page is a collection of community-made tools for the technical side of the game.

https://pdsls.dev/
PDSls

This site allows you to browse the AT Protocol's technical side. Personally, I don't really understand the technical details, but I think somebody else might do.

https://cheat.sh/
cheat.sh

cheat.sh is a collection of cheat sheets for various programming languages. This helped me on my way to become a 'nerd'.

https://wttr.in/
wttr.in

wttr.in is a browser/terminal weather service. It allows a surprising amount of customisation via URL parameters.

https://ipaddy.net/
IP Addy

Shows your IP address. Also works as an API.

https://marcan.github.io/wii-parental-reset-tool/
Wii Parental Controls Reset Tool

Allows you to reset the parental controls of the Wii console. Originally, you had to call Nintendo to get a reset code. This service (according to a Reddit post) still works, but this site is easier to use (and free).

Programming

https://github.com
Github

Github is a site where you can program, share your creations and see what others have done. It also offers a service called "Github Pages", which is a free website hosting service. (free as in "free beer")

https://val.town
Val Town

Val Town is similar to Github. The site in nutshell:

If Github Gists could run. And AWS Lambda were fun.

https://strudel.cc
Strudel

Strudel is a tool for live coding and music making. It uses a language called TidalCycles, which is easy to use.

https://compiler-explorer.com/
Compiler-Explorer

Allows you to run some (potentially weirder) programming languages in browser. Useful for C.

https://pseudorandom.name/
Pseudorandom Name Generator

Generates a pseudorandom English-language name.

https://www.random.org/integers/
Random.org (integers)

Generates one random integer, using the most foolproof RNG algorithm so far: atmospheric noise.

https://whatthecommit.org/
What The Commit

Generates a random commit message. Now you can push the .env file to public with style.

Privacy and Safety

https://haveibeenpwned.com/
Have I Been Pwned

Check if your emails were part of data breaches.

Wikis

https://wikipedia.org/
Wikipedia

Wikipedia is a free online encyclopedia available in many different languages. If you want an answer to something, it's probably here. If you speak a language, it has probably a Wikipedia.

  • English
  • Finnish
  • Swedish
  • Russian

https://wiibrew.org
WiiBrew

WiiBrew is a wiki for Wii homebrew. It also has all known technical details for the console.

https://wiki.bedrock.dev/
Minecraft Bedrock Wiki

A wiki for the technical side of Minecraft Bedrock Edition.

Wii Settings HTML

Wii Settings are just HTML. They are also language-specific, and run by the Opera browser.
